# rmqclient
RabbitMQ client example program using amqpcpp.

## Quick Start

Test environment:

- ubuntu 16.04
- cmake 3.5.1
- rabbitmq-server 3.5.7

Build:

```bash
# mkdir build && cd build
# cmake ../src && make
```
